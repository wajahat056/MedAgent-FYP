# MedAgent

Agentic AI system for plain-language explanation of medical lab reports.
BSc Final Year Project — University of Hertfordshire (6WCM0029-0509)

**Author:** Wajahat Maroof (SRN: 25002780)
**Supervisor:** Prapa Rattadilok

## Status
Project in early development. Ethics approval submitted and pending.

## Architecture
- Extraction Agent — parses PDF/image lab reports
- Analysis Agent — flags abnormal values
- RAG Agent — retrieves medical context (MedlinePlus, NHS Inform)
- Report Writer Agent — generates plain-English explanation
